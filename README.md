# Palindromy
Palindrom je slovo, které se čte stejně zleva doprava i zprava doleva. Například kajak nebo radar.
Upravte soubor app.js tak, aby prošel všechny položky li a ověřil, jestli se v jejich textovém obsahu nachází palindrom. Pokud ano, přidejte položce třídu palindrom, která má již přepřipravené CSS pro zvýraznění.
BONUS: Pro určení palindromů ignorujte velikost písmen a mezery ve větách. Předpokládejte tedy, že i Anna nebo drb z Brd jsou palindromy.
